# Волшебная школа медицинских инноваций им. Асклепия

Single-file build. Всё в одном `index.html`, картинка отдельно в `public/map-base.jpg`.

## Запуск локально

```bash
python3 -m http.server 8080
```

Открыть `http://localhost:8080/`. Через `file://` не работает — Babel standalone не загружает inline-скрипты по file-протоколу в некоторых браузерах.

## Деплой

Залить эту папку на любой статический хостинг (Vercel, Netlify, GitHub Pages). Файл `vercel.json` отключает auto-detect фреймворка.

## Атмосферный эффект

Облака покрывают всю карту полупрозрачным движущимся слоем. Вокруг курсора (≈200px диаметр) облака рассеиваются — открывая чистый фрагмент карты с плавным переходом по краям. На тач-устройствах (без курсора) этот эффект отключен, облака плывут поверх всей карты.

## Структура

```
.
├── index.html         # ВСЁ — HTML + CSS + React + все компоненты в одном файле
├── public/
│   └── map-base.jpg   # карта 3168×1344 (1.1 MB)
└── vercel.json
```
